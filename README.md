# PCL2-KingSkin

#### 介绍

PCL2的自定义主题

#### 插件架构

插件为XML语言的xaml
感兴趣的大佬去看看成品提交反馈

#### 使用教程

1.  解压PCL.zip到任意文件夹
2.  打开PCL.exe（启动器本体）按图标开始Minecraft之旅
