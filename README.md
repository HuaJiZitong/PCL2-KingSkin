# PCL2-KingSkin

#### 介绍

PCL2的自定义主题项目

#### 皮肤插件框架

插件为XML语言的xaml
感兴趣的大佬去看看作品提交Requests

#### 使用教程

1.  解压PCL.zip到任意文件夹
2.  打开PCL.exe（启动器本体）
3.  皮肤插件自动加载
4.  开始Minecraft之旅
